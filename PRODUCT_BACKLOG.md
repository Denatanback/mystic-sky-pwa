# eLuna Product Backlog

## Phase 2 / Beta: AI Oracle Assistant

Status: Deferred

Reason:
We decided not to include Oracle in the alpha build to avoid extra API dependency, cost control complexity, and safety/QA scope before the first user test.

Future scope:
- /oracle page
- contextual assistant for daily reading, card, path, practices
- Free: 1 question/day
- Trial/Premium: increased daily limits
- server-only OpenAI integration
- no API key on client
- safety guardrails
- contextual answers based on profile, daily card, path, preland result

Recommended model later:
- cheap default model for most answers
- stricter daily limits
- no AI generation on page load
- AI only after user action
